## 精读摘要

### MemPrivacy: Privacy-Preserving Personalized Memory Management for Edge-Cloud Agents
- 类型：论文
- 优先级：高
- 链接：https://huggingface.co/papers/2605.09530
- 一句话：为边缘-云代理提供隐私保护个性化记忆管理

**它解决什么问题**

云端记忆管理暴露用户隐私，现有方法过度屏蔽导致语义丢失

**核心创新**

1. 在边缘设备识别隐私敏感片段并替换为类型化占位符
2. 构建包含200用户5.2万隐私实例的四级隐私分类基准
3. 将隐私保护与语义解耦，效用损失控制在1.6%以内

**为什么重要**

平衡隐私保护与记忆效用，使边缘-云代理能在安全前提下实现个性化长期适应

**需要知道的基础知识**

LLM代理在边缘-云环境部署需要个性化记忆，但隐私保护与语义保留存在矛盾

**后续追踪**

该方法在更多记忆系统和实际部署场景中的泛化能力

### RubricEM: Meta-RL with Rubric-guided Policy Decomposition beyond Verifiable Rewards
- 类型：论文
- 优先级：高
- 链接：https://huggingface.co/papers/2605.10899
- 一句话：基于评分标准的元强化学习框架，用于深度研究代理的策略分解

**它解决什么问题**

深度研究代理的输出缺乏可验证奖励，长轨迹难以分配信用和复用经验

**核心创新**

1. 以评分标准作为策略执行、评判反馈和代理记忆的共享接口
2. 提出阶段式策略分解与基于反思的元策略演化
3. 在4个长文研究基准上超越同等规模开源模型

**为什么重要**

为无法用简单奖励驱动的复杂研究任务提供了可扩展的强化学习训练范式

**需要知道的基础知识**

现有强化学习依赖可验证奖励，但深度研究代理的开放输出无法直接评分

**后续追踪**

该框架在更多工具使用和科学发现场景中的适用性

### ToolCUA: Towards Optimal GUI-Tool Path Orchestration for Computer Use Agents
- 类型：论文
- 优先级：高
- 链接：https://huggingface.co/papers/2605.12481
- 一句话：端到端学习GUI与工具混合操作路径选择的计算机使用代理

**它解决什么问题**

计算机使用代理在GUI操作和高级工具调用之间决策困难，缺乏高质量混合轨迹

**核心创新**

1. 提出交错GUI-工具轨迹扩增流水线，重用静态GUI轨迹
2. 设计工具引导的GUI RFT和在线代理RL训练
3. 在OSWorld-MCP上达到46.85%准确率，相对提升66%

**为什么重要**

实现了GUI和工具动作的高效编排，为真实世界数字代理提供了可行范式

**需要知道的基础知识**

现有计算机使用代理通常只使用GUI或只使用工具，但许多任务需要两者结合

**后续追踪**

该方法在更多平台和长尾任务上的泛化能力

### LongMemEval-V2: Evaluating Long-Term Agent Memory Toward Experienced Colleagues
- 类型：benchmark
- 优先级：高
- 链接：https://arxiv.org/abs/2605.12493v1
- 一句话：评估长期代理记忆在定制化环境中的能力基准，含451个问题

**它解决什么问题**

现有记忆基准多关注用户历史或短轨迹，缺乏对代理环境经验内化的直接评估

**核心创新**

1. 包含静态状态回忆、动态状态跟踪、工作流知识等5种核心记忆能力
2. 提出AgentRunbook-C方法，依赖编码代理收集证据
3. 在115M token轨迹上评估，最优方法准确率72.5%，远超RAG基线

**为什么重要**

为构建能积累环境经验的长期代理记忆系统提供了标准化评估

**需要知道的基础知识**

代理在专用网络环境中需要长期记忆来记住界面、状态、工作流和常见错误

**后续追踪**

编码代理方法的延迟优化及更多记忆架构的融合
