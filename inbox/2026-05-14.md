## 精读摘要

### MemPrivacy: Privacy-Preserving Personalized Memory Management for Edge-Cloud Agents
- 类型：论文
- 优先级：高
- 链接：https://huggingface.co/papers/2605.09530
- 一句话：提出MemPrivacy，保护边缘-云AI agent个性化记忆的隐私，平衡隐私与效用。

**它解决什么问题**

边缘-云agent的个性化记忆暴露敏感信息，现有隐私保护方法过度遮蔽导致记忆效用下降。

**核心创新**

1. 边缘设备识别隐私敏感跨度并替换为类型化占位符
2. 云端记忆处理，本地恢复原文，解耦隐私保护与语义破坏
3. 构建MemPrivacy-Bench，含200用户52k隐私实例及四级隐私分类

**为什么重要**

在边缘-云agent部署中实现隐私与效用平衡，是AI agent安全落地的关键支撑。

**需要知道的基础知识**

了解LLM agent在边缘-云环境中的记忆管理及常见隐私保护方法。

**后续追踪**

MemPrivacy在更多记忆系统和真实场景中的泛化表现。

### RubricEM: Meta-RL with Rubric-guided Policy Decomposition beyond Verifiable Rewards
- 类型：论文
- 优先级：高
- 链接：https://huggingface.co/papers/2605.10899
- 一句话：提出RubricEM，用规则引导策略分解训练深度研究agent，超越可验证奖励。

**它解决什么问题**

深度研究agent输出无标准答案、轨迹长，标准RL无法使用可验证奖励进行后训练。

**核心创新**

1. 规则作为策略执行、评判反馈和agent记忆的共享接口
2. 阶段式策略分解结合基于反思的元策略进化
3. Stage-Structured GRPO提供密集语义反馈实现长程优化

**为什么重要**

显著提升长格式研究agent性能，RubricEM-8B接近专有深度研究系统。

**需要知道的基础知识**

了解强化学习中针对不可验证奖励任务的后训练方法。

**后续追踪**

RubricEM在更多复杂推理任务上的应用与扩展。

### ToolCUA: Towards Optimal GUI-Tool Path Orchestration for Computer Use Agents
- 类型：论文
- 优先级：高
- 链接：https://huggingface.co/papers/2605.12481
- 一句话：提出ToolCUA，学习计算机使用agent在GUI和工具间的最优切换路径。

**它解决什么问题**

计算机使用agent在混合动作空间中不知何时切换GUI与工具，导致执行路径次优。

**核心创新**

1. 交错GUI-工具轨迹扩缩流水线，复用静态GUI轨迹合成工具库
2. 工具引导的GUI RFT，结合SFT和单轮RL优化切换点
3. 在线agent RL及工具高效路径奖励，鼓励适当工具使用

**为什么重要**

在OSWorld-MCP达46.85%准确率（相对提升66%），验证混合动作空间训练范式有效。

**需要知道的基础知识**

了解计算机使用agent、GUI原子动作与API工具的概念。

**后续追踪**

ToolCUA在真实软件环境中的泛化能力及延迟优化。

### LongMemEval-V2: Evaluating Long-Term Agent Memory Toward Experienced Colleagues
- 类型：benchmark
- 优先级：高
- 链接：https://arxiv.org/abs/2605.12493v1
- 一句话：推出LongMemEval-V2基准，评估web agent长期记忆，含451个手工问题。

**它解决什么问题**

现有记忆基准聚焦短轨迹或下游任务，缺乏对agent内化环境经验的直接评估。

**核心创新**

1. 覆盖静态状态回忆、动态状态跟踪等5种核心记忆能力
2. 历史轨迹达500条、1.15亿token，规模大
3. 提出AgentRunbook-R和AgentRunbook-C两种记忆方法

**为什么重要**

提供web agent长期记忆的系统评估工具箱，推动记忆系统研究。

**需要知道的基础知识**

了解agent记忆定义、web环境交互及现有记忆基准。

**后续追踪**

AgentRunbook方法的延迟-精度前沿改进及更广泛应用。
